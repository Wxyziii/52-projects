# Week 37 - Screenshot Utility

## English

### Overview
A screenshot capture utility that takes screenshots of the entire screen, specific windows, or custom regions. Includes editing features and various save formats for flexible screen capturing.

### Features
- **Full Screen Capture**: Capture entire display
- **Window Capture**: Screenshot specific windows
- **Region Selection**: Drag to select custom area
- **Hotkey Support**: Keyboard shortcuts for quick capture
- **Annotations**: Add text, arrows, shapes
- **Save Formats**: PNG, JPG, BMP support
- **Clipboard Copy**: Direct copy to clipboard

### Technical Details
- **Language**: C++
- **APIs**: Windows GDI+, X11 XGetImage, macOS Core Graphics
- **Image Processing**: Pixel manipulation and encoding
- **GUI**: Overlay for region selection

### Use Cases
- Screen documentation
- Bug reporting
- Tutorial creation
- Learning graphics APIs
- Building productivity tools

## Norsk

### Oversikt
Et skjermbildeverktøy som tar skjermbilder av hele skjermen, spesifikke vinduer eller tilpassede regioner. Inkluderer redigeringsfunksjoner og ulike lagringsformater for fleksibel skjermfangst.

### Funksjoner
- **Fullskjermfangst**: Fang hele displayet
- **Vindusfangst**: Skjermbilde spesifikke vinduer
- **Områdeutvelgelse**: Dra for å velge tilpasset område
- **Hurtigtast-støtte**: Tastatursnarveier for rask fangst
- **Annotasjoner**: Legg til tekst, piler, former
- **Lagringsformater**: PNG, JPG, BMP-støtte
- **Utklippstavlekopiering**: Direkte kopiering til utklippstavle

### Tekniske detaljer
- **Språk**: C++
- **APIer**: Windows GDI+, X11 XGetImage, macOS Core Graphics
- **Bildebehandling**: Pikselmanipulering og koding
- **GUI**: Overlay for områdeutvelgelse

### Bruksområder
- Skjermdokumentasjon
- Feilrapportering
- Opplæringsmateriellopprettelse
- Læring av grafikk-APIer
- Bygging av produktivitetsverktøy
