# Week 16 - Expression Parser Evaluator

## English

### Overview
A mathematical expression parser and evaluator that converts infix notation (like "2 + 3 * 4") to an evaluable form and computes results. Implements operator precedence, parentheses, and various mathematical functions.

### Features
- **Infix Parsing**: Parse standard mathematical notation
- **Operator Precedence**: Correct order of operations
- **Parentheses Support**: Nested expression evaluation
- **Functions**: sin, cos, sqrt, log, etc.
- **Variables**: Support for named variables
- **Error Detection**: Syntax error reporting

### Technical Details
- **Language**: C++
- **Algorithm**: Shunting Yard or Recursive Descent
- **Data Structures**: Stacks for operators and operands
- **Conversion**: Infix to Postfix (RPN) conversion

### Use Cases
- Calculator applications
- Spreadsheet formula evaluation
- Mathematical software
- Learning parsing techniques
- Compiler theory fundamentals

## Norsk

### Oversikt
En matematisk uttrykksparser og evaluator som konverterer infix-notasjon (som "2 + 3 * 4") til en evaluerbar form og beregner resultater. Implementerer operatørprioritet, parenteser og ulike matematiske funksjoner.

### Funksjoner
- **Infix-parsing**: Parse standard matematisk notasjon
- **Operatørprioritet**: Korrekt rekkefølge av operasjoner
- **Parentesstøtte**: Nestet uttrykks-evaluering
- **Funksjoner**: sin, cos, sqrt, log, etc.
- **Variabler**: Støtte for navngitte variabler
- **Feildeteksjon**: Syntaksfeilrapportering

### Tekniske detaljer
- **Språk**: C++
- **Algoritme**: Shunting Yard eller Recursive Descent
- **Datastrukturer**: Stabler for operatører og operander
- **Konvertering**: Infix til Postfix (RPN) konvertering

### Bruksområder

- Regnearkformel-evaluering
