# Week 27 - Simple HTTP Server

## English

### Overview
A simple HTTP server that serves static files and handles basic HTTP requests. Demonstrates server-side network programming, HTTP protocol implementation, and request routing.

### Features
- **Static File Serving**: Serve HTML, CSS, JS, images
- **HTTP Request Handling**: Parse and respond to requests
- **MIME Types**: Proper content-type headers
- **Directory Listing**: Browse directory structure
- **Multiple Connections**: Handle concurrent requests
- **Logging**: Access logs for requests
- **Error Pages**: 404, 500 error responses

### Technical Details
- **Language**: C++
- **Protocol**: HTTP/1.1
- **Sockets**: TCP server socket
- **Threading**: Multi-threaded request handling

### Use Cases
- Learning server programming
- Development web server
- Understanding HTTP protocol
- Building web applications
- Static site hosting

## Norsk

### Oversikt
En enkel HTTP-server som serverer statiske filer og håndterer grunnleggende HTTP-forespørsler. Demonstrerer serversideprogrammering, HTTP-protokollimplementering og forespørselsruting.

### Funksjoner
- **Statisk filservering**: Server HTML, CSS, JS, bilder
- **HTTP-forespørselshåndtering**: Parse og svar på forespørsler
- **MIME-typer**: Riktige content-type-headere
- **Katalogliste**: Bla gjennom katalogstruktur
- **Flere tilkoblinger**: Håndter samtidige forespørsler
- **Logging**: Tilgangslogger for forespørsler
- **Feilsider**: 404, 500-feilrespons

### Tekniske detaljer
- **Språk**: C++
- **Protokoll**: HTTP/1.1
- **Sockets**: TCP server socket
- **Tråding**: Flertrådet forespørselshåndtering

### Bruksområder
- Læring av serverprogrammering
- Utviklings-webserver
- Forståelse av HTTP-protokoll
- Bygging av webapplikasjoner
- Statisk nettstedshosting
