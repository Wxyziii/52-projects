# Week 32 - Network Port Scanner

## English

### Overview
A network port scanner that checks which ports are open on a target host. Useful for network diagnostics, security auditing, and understanding network services. Similar to nmap but custom-built.

### Features
- **Port Scanning**: Check TCP/UDP port status
- **Range Scanning**: Scan multiple ports or port ranges
- **Service Detection**: Identify common services on ports
- **Multi-threading**: Fast parallel scanning
- **Timeout Configuration**: Adjustable connection timeouts
- **Output Formats**: Text, JSON reporting

### Technical Details
- **Language**: C++
- **Protocol**: TCP/UDP socket connections
- **Threading**: Parallel port checks
- **Detection**: Banner grabbing for service identification

### Use Cases
- Network diagnostics
- Security auditing
- Learning network protocols
- Understanding port scanning
- System administration

## Norsk

### Oversikt
En nettverksport-skanner som sjekker hvilke porter som er åpne på en målvert. Nyttig for nettverksdiagnostikk, sikkerhetsrevisjon og forståelse av nettverkstjenester. Lik nmap, men tilpasset bygget.

### Funksjoner
- **Portskanning**: Sjekk TCP/UDP-portstatus
- **Områdeskanning**: Skann flere porter eller portområder
- **Tjenestedeteksjon**: Identifiser vanlige tjenester på porter
- **Multi-threading**: Rask parallell skanning
- **Timeout-konfigurasjon**: Justerbare tilkoblings-timeouts
- **Utdataformater**: Tekst, JSON-rapportering

### Tekniske detaljer
- **Språk**: C++
- **Protokoll**: TCP/UDP socket-tilkoblinger
- **Tråding**: Parallelle portsjekker
- **Deteksjon**: Banner-grabbing for tjenesteidentifikasjon

### Bruksområder
- Nettverksdiagnostikk
- Sikkerhetsrevisjon
- Læring av nettverksprotokoller
- Forståelse av portskanning
- Systemadministrasjon
