# Week 05 - Linked List Stack Queue Library

## English

### Overview
A comprehensive library implementing fundamental data structures: linked lists, stacks, and queues. This project provides reusable implementations of these essential data structures with full CRUD operations.

### Features
- **Singly Linked List**: Basic linked list with insert, delete, search
- **Doubly Linked List**: Bidirectional traversal support
- **Stack (LIFO)**: Push, pop, peek operations
- **Queue (FIFO)**: Enqueue, dequeue, front/back access
- **Generic Implementation**: Template-based for any data type
- **Memory Safe**: Proper memory management and cleanup

### Technical Details
- **Language**: C++
- **Templates**: Generic implementation for type flexibility
- **Memory**: Dynamic allocation with proper destructor handling
- **Complexity**: O(1) for stack/queue operations, O(n) for list search

### Use Cases
- Learning fundamental data structures
- Building block for complex algorithms
- Interview preparation
- Understanding pointer manipulation
- Foundation for more advanced structures (trees, graphs)

## Norsk

### Oversikt
Et omfattende bibliotek som implementerer grunnleggende datastrukturer: lenkede lister, stabler og køer. Dette prosjektet gir gjenbrukbare implementeringer av disse essensielle datastrukturene med fulle CRUD-operasjoner.

### Funksjoner
- **Enkelt lenket liste**: Grunnleggende lenket liste med innsetting, sletting, søk
- **Dobbelt lenket liste**: Støtte for toveis traversering
- **Stabel (LIFO)**: Push, pop, peek-operasjoner
- **Kø (FIFO)**: Enqueue, dequeue, front/back-tilgang
- **Generisk implementering**: Malbasert for alle datatyper
- **Minnesikker**: Riktig minnehåndtering og opprydding

### Tekniske detaljer
- **Språk**: C++
- **Maler**: Generisk implementering for typefleksibilitet
- **Minne**: Dynamisk allokering med riktig destruktorhåndtering
- **Kompleksitet**: O(1) for stabel/kø-operasjoner, O(n) for listesøk

### Bruksområder
- Læring av grunnleggende datastrukturer
- Byggestein for komplekse algoritmer
- Intervjuforberedelse
- Forståelse av peker-manipulering
- Grunnlag for mer avanserte strukturer (trær, grafer)
