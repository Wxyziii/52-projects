# Week 06 - Simple JSON Parser

## English

### Overview
A JSON parser built from scratch that reads, parses, and validates JSON data. This project demonstrates lexical analysis, syntax parsing, and data structure mapping without relying on external JSON libraries.

### Features
- **JSON Parsing**: Parse JSON strings into usable data structures
- **Validation**: Check JSON syntax and structure validity
- **Data Extraction**: Access values by key/index
- **Error Handling**: Detailed error messages for invalid JSON
- **Support for Types**: Objects, arrays, strings, numbers, booleans, null
- **Pretty Printing**: Format JSON output for readability

### Technical Details
- **Language**: C++
- **Parsing Method**: Recursive descent or tokenization
- **Data Structures**: Maps and vectors for internal representation
- **No External Dependencies**: Pure C++ implementation

### Use Cases
- Understanding parsing techniques
- Learning compiler/interpreter fundamentals
- Configuration file processing
- API response handling
- Educational tool for teaching parsing concepts

## Norsk

### Oversikt
En JSON-parser bygget fra bunnen av som leser, parser og validerer JSON-data. Dette prosjektet demonstrerer leksikalsk analyse, syntaksanalyse og datastruktur-mapping uten å stole på eksterne JSON-biblioteker.

### Funksjoner
- **JSON-parsing**: Parse JSON-strenger til brukbare datastrukturer
- **Validering**: Sjekk JSON-syntaks og strukturgyldighet
- **Dataekstraksjon**: Få tilgang til verdier etter nøkkel/indeks
- **Feilhåndtering**: Detaljerte feilmeldinger for ugyldig JSON
- **Støtte for typer**: Objekter, arrays, strenger, tall, boolske verdier, null
- **Pen utskrift**: Formater JSON-output for lesbarhet

### Tekniske detaljer
- **Språk**: C++
- **Parsemetode**: Rekursiv nedstigning eller tokenisering
- **Datastrukturer**: Maps og vektorer for intern representasjon
- **Ingen eksterne avhengigheter**: Ren C++-implementering

### Bruksområder
- Forståelse av parseteknikker
- Læring av kompilator/tolk-fundamenter
- Behandling av konfigurasjonsfiler
- API-respons håndtering
- Undervisningsverktøy for å lære parsekonsepter
