# Week 35 - Password Generator GUI

## English

### Overview
A graphical password generator application with customizable options for creating strong, random passwords. User-friendly interface for generating secure passwords with various character sets and lengths.

### Features
- **Customizable Length**: User-defined password length
- **Character Sets**: Uppercase, lowercase, numbers, symbols
- **Multiple Passwords**: Generate several at once
- **Strength Indicator**: Visual password strength meter
- **Copy to Clipboard**: One-click copying
- **Password History**: Recent passwords list
- **Exclude Similar Characters**: Avoid confusing characters

### Technical Details
- **Language**: C++
- **GUI Framework**: Qt, wxWidgets, or similar
- **Randomness**: Cryptographically secure random (std::random_device)
- **Entropy Calculation**: Password strength analysis

### Use Cases
- Creating secure passwords
- Learning GUI development
- Understanding password security
- Building security tools
- User-friendly security applications

## Norsk

### Oversikt
En grafisk passordgeneratorapplikasjon med tilpassbare alternativer for å lage sterke, tilfeldige passord. Brukervennlig grensesnitt for generering av sikre passord med ulike tegnsett og lengder.

### Funksjoner
- **Tilpassbar lengde**: Brukerdefinert passordlengde
- **Tegnsett**: Store bokstaver, små bokstaver, tall, symboler
- **Flere passord**: Generer flere samtidig
- **Styrkeindikator**: Visuell passordstyrke-måler
- **Kopier til utklippstavle**: Ett-klikks kopiering
- **Passordhistorikk**: Liste over nylige passord
- **Ekskluder lignende tegn**: Unngå forvirrende tegn

### Tekniske detaljer
- **Språk**: C++
- **GUI-rammeverk**: Qt, wxWidgets, eller lignende
- **Tilfeldighet**: Kryptografisk sikker random (std::random_device)
- **Entropiberegning**: Passordstyrkeanalyse

### Bruksområder
- Oppretting av sikre passord
- Læring av GUI-utvikling
- Forståelse av passordsikkerhet
- Bygging av sikkerhetsverktøy
- Brukervennlige sikkerhetsapplikasjoner
