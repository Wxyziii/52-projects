# Week 25 - TCP Chat Server Client

## English

### Overview
A TCP-based chat application with server and client components. Multiple clients can connect to a server and exchange messages in real-time, demonstrating network programming and socket communication.

### Features
- **Server-Client Architecture**: Central server with multiple clients
- **Real-time Messaging**: Instant message delivery
- **Multi-client Support**: Handle multiple connections simultaneously
- **Username System**: Identify users by names
- **Broadcast Messages**: Send to all connected clients
- **Private Messages**: Direct messaging between users

### Technical Details
- **Language**: C++
- **Protocol**: TCP/IP sockets
- **Concurrency**: Multi-threaded server (one thread per client)
- **Platform**: Cross-platform sockets (Winsock/BSD sockets)

### Use Cases
- Learning network programming
- Understanding client-server architecture
- Real-time communication systems
- Building chat applications
- Understanding socket programming

## Norsk

### Oversikt
En TCP-basert chat-applikasjon med server- og klientkomponenter. Flere klienter kan koble til en server og utveksle meldinger i sanntid, demonstrerer nettverksprogrammering og socket-kommunikasjon.

### Funksjoner
- **Server-klient-arkitektur**: Sentral server med flere klienter
- **Sanntidsmeldinger**: Øyeblikkelig meldingslevering
- **Flerklient-støtte**: Håndter flere tilkoblinger samtidig
- **Brukernavnsystem**: Identifiser brukere etter navn
- **Kringkastingsmeldinger**: Send til alle tilkoblede klienter
- **Private meldinger**: Direkte meldinger mellom brukere

### Tekniske detaljer
- **Språk**: C++
- **Protokoll**: TCP/IP-sockets
- **Samtidighet**: Flertrådet server (en tråd per klient)
- **Plattform**: Kryssplattform sockets (Winsock/BSD sockets)

### Bruksområder
- Læring av nettverksprogrammering
- Forståelse av klient-server-arkitektur
- Sanntidskommunikasjonssystemer
- Bygging av chat-applikasjoner
- Forståelse av socket-programmering
