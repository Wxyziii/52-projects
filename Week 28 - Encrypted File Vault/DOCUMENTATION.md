# Week 28 - Encrypted File Vault

## English

### Overview
A secure file storage system that encrypts files before storage and decrypts them on access. Protects sensitive data using industry-standard encryption algorithms like AES.

### Features
- **File Encryption**: Encrypt files with strong algorithms (AES-256)
- **File Decryption**: Restore original files with password
- **Password Protection**: Secure access control
- **Vault Management**: Add, remove, list encrypted files
- **Key Derivation**: PBKDF2 or similar for key generation
- **Secure Deletion**: Overwrite original files

### Technical Details
- **Language**: C++
- **Encryption**: AES-256 or ChaCha20
- **Libraries**: OpenSSL, libsodium, or Crypto++
- **Key Management**: Secure key storage and derivation

### Use Cases
- Protecting sensitive documents
- Learning cryptography
- Secure file storage
- Understanding encryption
- Privacy applications

## Norsk

### Oversikt
Et sikkert fillagringssystem som krypterer filer før lagring og dekrypterer dem ved tilgang. Beskytter sensitiv data ved hjelp av bransjestandard krypteringsalgoritmer som AES.

### Funksjoner
- **Filkryptering**: Krypter filer med sterke algoritmer (AES-256)
- **Fildekryptering**: Gjenopprett originale filer med passord
- **Passordbeskyttelse**: Sikker tilgangskontroll
- **Vault-håndtering**: Legg til, fjern, list krypterte filer
- **Nøkkelderivering**: PBKDF2 eller lignende for nøkkelgenerering
- **Sikker sletting**: Overskriv originale filer

### Tekniske detaljer
- **Språk**: C++
- **Kryptering**: AES-256 eller ChaCha20
- **Biblioteker**: OpenSSL, libsodium, eller Crypto++
- **Nøkkelhåndtering**: Sikker nøkkellagring og derivering

### Bruksområder
- Beskyttelse av sensitive dokumenter
- Læring av kryptografi
- Sikker fillagring
- Forståelse av kryptering
- Personvernapplikasjoner
