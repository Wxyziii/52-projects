# Week 14 - Benchmarking Framework

## English

### Overview
A performance benchmarking framework that measures execution time, memory usage, and other metrics for code snippets or functions. Essential for performance optimization and comparing algorithm implementations.

### Features
- **Time Measurement**: High-precision timing using chrono
- **Memory Profiling**: Track memory allocation and usage
- **Statistical Analysis**: Average, median, standard deviation
- **Comparison Mode**: Compare multiple implementations
- **Warm-up Rounds**: Eliminate cold start effects
- **Report Generation**: Detailed performance reports

### Technical Details
- **Language**: C++
- **Timing**: std::chrono for microsecond precision
- **Memory**: Custom allocator tracking or system calls
- **Statistics**: Mathematical analysis of results

### Use Cases
- Algorithm performance testing
- Code optimization
- Regression testing
- Comparing implementations
- Learning performance analysis

## Norsk

### Oversikt
Et ytelsesbenmarkingrammeverk som måler utførelsestid, minnebruk og andre målinger for kodesnutter eller funksjoner. Essensielt for ytelseoptimalisering og sammenligning av algoritmeimplementeringer.

### Funksjoner
- **Tidsmåling**: Høypresisjon timing ved bruk av chrono
- **Minneprofilering**: Spor minneallokering og bruk
- **Statistisk analyse**: Gjennomsnitt, median, standardavvik
- **Sammenligningsmodus**: Sammenlign flere implementeringer
- **Oppvarmingsrunder**: Eliminere kaldstarteffekter
- **Rapportgenerering**: Detaljerte ytelsesrapporter

### Tekniske detaljer
- **Språk**: C++
- **Timing**: std::chrono for mikrosekund presisjon
- **Minne**: Tilpasset allokeringssporing eller systemanrop
- **Statistikk**: Matematisk analyse av resultater

### Bruksområder
- Algoritmeytelsetesting
- Kodeoptimalisering
- Regresjonstesting
- Sammenligning av implementeringer
- Læring av ytelsesanalyse
