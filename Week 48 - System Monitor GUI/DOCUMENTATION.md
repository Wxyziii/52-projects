# Week 48 - System Monitor GUI

## English

### Overview
A graphical system monitoring application that displays real-time CPU, memory, disk, and network usage. Provides visual charts and detailed system information for comprehensive system monitoring.

### Features
- **Real-time Monitoring**: Live system metrics
- **CPU Usage**: Per-core utilization graphs
- **Memory Stats**: RAM and swap usage
- **Disk Activity**: I/O operations and space
- **Network Traffic**: Upload/download speeds
- **Process List**: Detailed process information
- **Historical Data**: Charts showing trends
- **Alerts**: Threshold-based notifications

### Technical Details
- **Language**: C++
- **GUI Framework**: Qt or wxWidgets
- **Charting**: QCustomPlot or similar
- **System APIs**: Platform-specific system calls
- **Threading**: Non-blocking UI updates

### Use Cases
- System performance monitoring
- Resource usage tracking
- Troubleshooting performance issues
- Learning system programming
- Building monitoring tools

## Norsk

### Oversikt
En grafisk systemovervåkingsapplikasjon som viser sanntids CPU-, minne-, disk- og nettverksbruk. Gir visuelle diagrammer og detaljert systeminformasjon for omfattende systemovervåking.

### Funksjoner
- **Sanntidsovervåking**: Live systemmålinger
- **CPU-bruk**: Per-kjerne utnyttelsesdiagrammer
- **Minnestatistikk**: RAM og swap-bruk
- **Diskaktivitet**: I/O-operasjoner og plass
- **Nettverkstrafikk**: Opplastings/nedlastingshastigheter
- **Prosessliste**: Detaljert prosessinformasjon
- **Historiske data**: Diagrammer som viser trender
- **Varsler**: Terskelbaserte varsler

### Tekniske detaljer
- **Språk**: C++
- **GUI-rammeverk**: Qt eller wxWidgets
- **Diagrammer**: QCustomPlot eller lignende
- **System-APIer**: Plattformspesifikke systemanrop
- **Tråding**: Ikke-blokkerende UI-oppdateringer

### Bruksområder
- Systemytelsesovervåking
- Ressursbrukssporing
- Feilsøking av ytelsesproblemer
- Læring av systemprogrammering
- Bygging av overvåkingsverktøy
