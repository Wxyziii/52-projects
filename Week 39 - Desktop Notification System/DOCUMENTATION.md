# Week 39 - Desktop Notification System

## English

### Overview
A desktop notification system that displays pop-up notifications for events, reminders, or alerts. Integrates with system notification APIs for native-looking notifications across platforms.

### Features
- **Custom Notifications**: Display messages with icons
- **Action Buttons**: Interactive notification buttons
- **Notification Queue**: Manage multiple notifications
- **Scheduling**: Delayed or scheduled notifications
- **Sound Alerts**: Audio notification support
- **Persistent/Transient**: Configurable notification duration
- **System Integration**: Native OS notification style

### Technical Details
- **Language**: C++
- **APIs**: Windows Toast, Linux libnotify, macOS NSUserNotificationCenter
- **GUI**: System-native notification display
- **Scheduling**: Timer-based notification triggers

### Use Cases
- Reminder applications
- System alerts
- Application notifications
- Learning OS integration
- Building notification services

## Norsk

### Oversikt
Et desktop-varslingssystem som viser popup-varsler for hendelser, påminnelser eller advarsler. Integrerer med system-varslings-APIer for innfødt-utseende varsler på tvers av plattformer.

### Funksjoner
- **Tilpassede varsler**: Vis meldinger med ikoner
- **Handlingsknapper**: Interaktive varslingsknapper
- **Varslingskø**: Håndter flere varsler
- **Planlegging**: Forsinkede eller planlagte varsler
- **Lydvarsler**: Støtte for lydvarsling
- **Vedvarende/Midlertidig**: Konfigurerbar varslingsvarighet
- **Systemintegrasjon**: Innfødt OS-varslingsstil

### Tekniske detaljer
- **Språk**: C++
- **APIer**: Windows Toast, Linux libnotify, macOS NSUserNotificationCenter
- **GUI**: System-innfødt varslingsvisning
- **Planlegging**: Timer-baserte varslingsutløsere

### Bruksområder
- Påminnelsesapplikasjoner
- Systemvarsler
- Applikasjonsvarsler
- Læring av OS-integrasjon
- Bygging av varslingstjenester
