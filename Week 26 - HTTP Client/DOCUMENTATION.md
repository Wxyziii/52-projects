# Week 26 - HTTP Client

## English

### Overview
An HTTP client library that makes HTTP requests (GET, POST, PUT, DELETE) to web servers. Demonstrates understanding of the HTTP protocol, request/response handling, and web communication.

### Features
- **HTTP Methods**: GET, POST, PUT, DELETE support
- **Headers Management**: Custom request headers
- **Request Body**: Send data with POST/PUT
- **Response Parsing**: Parse status codes, headers, body
- **URL Parsing**: Break down URLs into components
- **Error Handling**: Handle connection and HTTP errors
- **Redirect Support**: Follow HTTP redirects

### Technical Details
- **Language**: C++
- **Protocol**: HTTP/1.1 (optionally HTTP/2)
- **Sockets**: TCP socket communication
- **Parsing**: HTTP message parsing

### Use Cases
- API consumption
- Web scraping
- Learning HTTP protocol
- Building REST clients
- Understanding web communication

## Norsk

### Oversikt
Et HTTP-klientbibliotek som gjør HTTP-forespørsler (GET, POST, PUT, DELETE) til webservere. Demonstrerer forståelse av HTTP-protokollen, forespørsel/respons-håndtering og webkommunikasjon.

### Funksjoner
- **HTTP-metoder**: GET, POST, PUT, DELETE-støtte
- **Headerhåndtering**: Tilpassede forespørselsheadere
- **Forespørselsbody**: Send data med POST/PUT
- **Responseparsing**: Parse statuskoder, headere, body
- **URL-parsing**: Del opp URLer i komponenter
- **Feilhåndtering**: Håndter tilkoblings- og HTTP-feil
- **Omdirigeringsstøtte**: Følg HTTP-omdirigeringer

### Tekniske detaljer
- **Språk**: C++
- **Protokoll**: HTTP/1.1 (valgfritt HTTP/2)
- **Sockets**: TCP socket-kommunikasjon
- **Parsing**: HTTP-meldingsparsing

### Bruksområder
- API-forbruk
- Web scraping
- Læring av HTTP-protokoll
- Bygging av REST-klienter
- Forståelse av webkommunikasjon
