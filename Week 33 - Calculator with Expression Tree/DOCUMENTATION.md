# Week 33 - Calculator with Expression Tree

## English

### Overview
An advanced calculator that uses expression trees to parse and evaluate mathematical expressions. Demonstrates tree data structures, parsing techniques, and computer algebra fundamentals.

### Features
- **Expression Trees**: Parse expressions into tree structure
- **Complex Operations**: +, -, *, /, ^, parentheses
- **Tree Visualization**: Display expression tree structure
- **Symbolic Differentiation**: Compute derivatives
- **Expression Simplification**: Reduce complex expressions
- **Variable Support**: Algebraic expressions with variables

### Technical Details
- **Language**: C++
- **Data Structure**: Binary expression tree
- **Parsing**: Recursive descent or operator precedence
- **Evaluation**: Tree traversal (postorder)

### Use Cases
- Learning tree data structures
- Understanding expression parsing
- Computer algebra systems
- Scientific calculator development
- Educational tool for teaching trees

## Norsk

### Oversikt
En avansert kalkulator som bruker uttrykkstrær for å parse og evaluere matematiske uttrykk. Demonstrerer tre-datastrukturer, parseteknikker og datamaskinalgbra-fundamenter.

### Funksjoner
- **Uttrykkstre**: Parse uttrykk til trestruktur
- **Komplekse operasjoner**: +, -, *, /, ^, parenteser
- **Tre-visualisering**: Vis uttrykkstrestruktur
- **Symbolsk differensiering**: Beregn derivater
- **Uttrykksforenkling**: Reduser komplekse uttrykk
- **Variabelstøtte**: Algebraiske uttrykk med variabler

### Tekniske detaljer
- **Språk**: C++
- **Datastruktur**: Binært uttrykkstré
- **Parsing**: Rekursiv nedstigning eller operatørprioritet
- **Evaluering**: Tre-traversering (postorder)

### Bruksområder
- Læring av tre-datastrukturer
- Forståelse av uttrykksparsing
- Datamaskinalgebrasystemer
- Vitenskapelig kalkulatorutvikling
- Undervisningsverktøy for å lære trær
