# Week 04 - Custom Dynamic Array

## English

### Overview
A custom implementation of a dynamic array data structure in C++, similar to `std::vector`. This project demonstrates low-level memory management, array resizing, and fundamental data structure concepts.

### Features
- **Dynamic Resizing**: Automatically grows when capacity is exceeded
- **Memory Management**: Manual allocation and deallocation
- **Basic Operations**: Push, pop, insert, delete elements
- **Index Access**: Fast random access via indexing
- **Capacity Management**: Efficient memory usage with growth strategies
- **Iterator Support**: Enable range-based operations

### Technical Details
- **Language**: C++
- **Memory Model**: Heap allocation with manual management
- **Growth Strategy**: Typically doubles capacity when full
- **Complexity**: O(1) amortized for push_back, O(1) for access

### Use Cases
- Understanding memory management in C++
- Learning data structure internals
- Performance-critical applications
- Educational purposes for teaching dynamic arrays
- Foundation for building custom containers

## Norsk

### Oversikt
En tilpasset implementering av en dynamisk array-datastruktur i C++, lik `std::vector`. Dette prosjektet demonstrerer lavnivå minnehåndtering, arrayendring og grunnleggende datastrukturkonsepter.

### Funksjoner
- **Dynamisk størrelse**: Vokser automatisk når kapasiteten overskrides
- **Minnehåndtering**: Manuell allokering og deallokering
- **Grunnleggende operasjoner**: Push, pop, sett inn, slett elementer
- **Indekstilgang**: Rask tilfeldig tilgang via indeksering
- **Kapasitetsstyring**: Effektiv minnebruk med vekststrategier
- **Iterator-støtte**: Muliggjør omfangsbaserte operasjoner

### Tekniske detaljer
- **Språk**: C++
- **Minnemodell**: Heap-allokering med manuell håndtering
- **Vekststrategi**: Dobler vanligvis kapasiteten når full
- **Kompleksitet**: O(1) amortisert for push_back, O(1) for tilgang

### Bruksområder
- Forståelse av minnehåndtering i C++
- Læring av datastruktur-internals
- Ytelseskritiske applikasjoner
- Utdanningsformål for undervisning i dynamiske arrays
- Grunnlag for bygging av tilpassede containere
