# Week 50 - Game Engine Skeleton

## English

### Overview
A basic game engine framework with core systems for rendering, input, audio, and game loop. Provides foundation for 2D or simple 3D games, demonstrating game architecture and engine design principles.

### Features
- **Game Loop**: Fixed timestep update loop
- **Rendering System**: 2D/3D graphics rendering
- **Input Handling**: Keyboard, mouse, gamepad support
- **Audio System**: Sound effects and music
- **Entity Component System**: Flexible game object architecture
- **Resource Management**: Load textures, models, sounds
- **Scene Management**: Multiple game scenes/levels
- **Physics Integration**: Basic collision detection

### Technical Details
- **Language**: C++
- **Graphics**: OpenGL, DirectX, or Vulkan
- **Audio**: OpenAL or FMOD
- **Libraries**: SDL2, GLFW for windowing
- **Architecture**: ECS or object-oriented design

### Use Cases
- Game development
- Learning engine architecture
- Graphics programming
- Understanding game systems
- Building custom games

## Norsk

### Oversikt
Et grunnleggende spillmotorrammeverk med kjernesystemer for rendering, input, lyd og spillsløyfe. Gir grunnlag for 2D eller enkle 3D-spill, demonstrerer spillarkitektur og motordesignprinsipper.

### Funksjoner
- **Spillsløyfe**: Fast tidssteg oppdateringssløyfe
- **Renderingssystem**: 2D/3D grafikk-rendering
- **Input-håndtering**: Tastatur, mus, gamepad-støtte
- **Lydsystem**: Lydeffekter og musikk
- **Entity Component System**: Fleksibel spillobjektarkitektur
- **Ressurshåndtering**: Last teksturer, modeller, lyder
- **Scenehåndtering**: Flere spillscener/nivåer
- **Fysikk-integrasjon**: Grunnleggende kollisjonsdeteksjon

### Tekniske detaljer
- **Språk**: C++
- **Grafikk**: OpenGL, DirectX, eller Vulkan
- **Lyd**: OpenAL eller FMOD
- **Biblioteker**: SDL2, GLFW for vinduer
- **Arkitektur**: ECS eller objektorientert design

### Bruksområder
- Spillutvikling
- Læring av motorarkitektur
- Grafikkprogrammering
- Forståelse av spillsystemer
- Bygging av tilpassede spill
