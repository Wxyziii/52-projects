# Week 34 - Cross-Platform GUI App

## English

### Overview
A graphical user interface application that works on multiple operating systems (Windows, Linux, macOS). Demonstrates cross-platform GUI development using frameworks like Qt, wxWidgets, or GTK.

### Features
- **Cross-Platform**: Single codebase for multiple OS
- **Native Look**: Platform-appropriate UI styling
- **Widgets**: Buttons, text fields, menus, dialogs
- **Event Handling**: User interaction management
- **Layouts**: Flexible window organization
- **File Dialogs**: Standard open/save dialogs

### Technical Details
- **Language**: C++
- **Framework**: Qt, wxWidgets, GTK, or Dear ImGui
- **Build System**: CMake for cross-platform builds
- **Packaging**: OS-specific installers

### Use Cases
- Desktop application development
- Learning GUI programming
- Cross-platform software
- Understanding UI frameworks
- Building user-facing tools

## Norsk

### Oversikt
En grafisk brukergrensesnitt-applikasjon som fungerer på flere operativsystemer (Windows, Linux, macOS). Demonstrerer kryssplattform GUI-utvikling ved hjelp av rammeverk som Qt, wxWidgets eller GTK.

### Funksjoner
- **Kryssplattform**: Én kodebase for flere OS
- **Innfødt utseende**: Plattform-passende UI-styling
- **Widgets**: Knapper, tekstfelt, menyer, dialoger
- **Hendelseshåndtering**: Brukerinteraksjonshåndtering
- **Layouts**: Fleksibel vindusorganisering
- **Fildialoger**: Standard åpne/lagre-dialoger

### Tekniske detaljer
- **Språk**: C++
- **Rammeverk**: Qt, wxWidgets, GTK, eller Dear ImGui
- **Byggesystem**: CMake for kryssplattform-bygg
- **Pakking**: OS-spesifikke installatører

### Bruksområder
- Desktop-applikasjonsutvikling
- Læring av GUI-programmering
- Kryssplattformprogramvare
- Forståelse av UI-rammeverk
- Bygging av brukerrettede verktøy
