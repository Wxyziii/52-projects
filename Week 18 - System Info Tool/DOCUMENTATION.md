# Week 18 - System Info Tool

## English

### Overview
A system information utility that gathers and displays comprehensive hardware and software details about the computer. Reports CPU, memory, disk, OS version, network interfaces, and more.

### Features
- **Hardware Info**: CPU model, cores, speed, memory capacity
- **OS Details**: Operating system, version, build number
- **Disk Information**: Storage devices, capacity, usage
- **Network Info**: IP addresses, network adapters
- **System Uptime**: How long the system has been running
- **Export Report**: Save information to file

### Technical Details
- **Language**: C++
- **System APIs**: Platform-specific system calls
- **Cross-platform**: Different implementations for Windows/Linux
- **WMI/sysfs**: Windows Management Instrumentation or Linux sysfs

### Use Cases
- System inventory
- Technical support diagnostics
- Hardware compatibility checking
- Learning system programming
- Understanding OS APIs

## Norsk

### Oversikt
Et systeminformasjonsverktøy som samler og viser omfattende maskinvare- og programvaredetaljer om datamaskinen. Rapporterer CPU, minne, disk, OS-versjon, nettverksgrensesnitt og mer.

### Funksjoner
- **Maskinvareinfo**: CPU-modell, kjerner, hastighet, minnekapasitet
- **OS-detaljer**: Operativsystem, versjon, byggnummer
- **Diskinformasjon**: Lagringsenheter, kapasitet, bruk
- **Nettverksinfo**: IP-adresser, nettverksadaptere
- **Systemoppetid**: Hvor lenge systemet har kjørt
- **Eksportrapport**: Lagre informasjon til fil

### Tekniske detaljer
- **Språk**: C++
- **System-APIer**: Plattformspesifikke systemanrop
- **Kryssplattform**: Forskjellige implementeringer for Windows/Linux
- **WMI/sysfs**: Windows Management Instrumentation eller Linux sysfs

### Bruksområder
- Systeminventar
- Teknisk støttediagnostikk
- Maskinvarekompatibilitetskontroll
- Læring av systemprogrammering
- Forståelse av OS-APIer
