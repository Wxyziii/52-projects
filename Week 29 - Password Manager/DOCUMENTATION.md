# Week 29 - Password Manager

## English

### Overview
A secure password management application that stores, generates, and manages passwords for various accounts. Encrypts password database with a master password for maximum security.

### Features
- **Password Storage**: Securely store website/app passwords
- **Password Generation**: Create strong random passwords
- **Master Password**: Single password to access vault
- **Encryption**: AES-256 encryption for database
- **Search Functionality**: Find passwords quickly
- **Password Strength Checker**: Evaluate password security
- **Auto-lock**: Timeout-based security

### Technical Details
- **Language**: C++
- **Encryption**: AES-256 with PBKDF2 key derivation
- **Storage**: Encrypted local database
- **Libraries**: Cryptographic libraries (OpenSSL, libsodium)

### Use Cases
- Personal password management
- Learning security best practices
- Understanding encryption
- Building security applications
- Privacy-focused tools

## Norsk

### Oversikt
En sikker passordbehandlingsapplikasjon som lagrer, genererer og håndterer passord for ulike kontoer. Krypterer passord-database med et hovedpassord for maksimal sikkerhet.

### Funksjoner
- **Passordlagring**: Lagre nettsted/app-passord sikkert
- **Passordgenerering**: Opprett sterke tilfeldige passord
- **Hovedpassord**: Ett passord for å få tilgang til vault
- **Kryptering**: AES-256-kryptering for database
- **Søkefunksjonalitet**: Finn passord raskt
- **Passordstyrkesjekker**: Evaluer passordsikkerhet
- **Auto-lås**: Timeout-basert sikkerhet

### Tekniske detaljer
- **Språk**: C++
- **Kryptering**: AES-256 med PBKDF2-nøkkelderivering
- **Lagring**: Kryptert lokal database
- **Biblioteker**: Kryptografiske biblioteker (OpenSSL, libsodium)

### Bruksområder
- Personlig passordbehandling
- Læring av beste sikkerhetspraksis
- Forståelse av kryptering
- Bygging av sikkerhetsapplikasjoner
- Personvernfokuserte verktøy
