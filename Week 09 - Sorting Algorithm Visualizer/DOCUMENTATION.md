# Week 09 - Sorting Algorithm Visualizer

## English

### Overview
A visual tool that demonstrates how different sorting algorithms work by showing step-by-step execution. Helps understand algorithm behavior, performance differences, and complexity through animated or console-based visualization.

### Features
- **Multiple Algorithms**: Bubble, Selection, Insertion, Merge, Quick Sort, etc.
- **Step-by-Step Visualization**: See each comparison and swap
- **Performance Metrics**: Track comparisons, swaps, and execution time
- **Comparison Mode**: Run multiple algorithms on same data
- **Custom Input**: Test with different array sizes and patterns
- **Speed Control**: Adjust visualization speed

### Technical Details
- **Language**: C++
- **Visualization**: Console-based or terminal graphics
- **Algorithms**: Various sorting algorithm implementations
- **Metrics**: Time complexity and operation counting

### Use Cases
- Learning sorting algorithms
- Algorithm performance comparison
- Teaching data structures and algorithms
- Understanding Big-O notation practically
- Interview preparation

## Norsk

### Oversikt
Et visuelt verktøy som demonstrerer hvordan forskjellige sorteringsalgoritmer fungerer ved å vise trinn-for-trinn-utførelse. Hjelper med å forstå algoritmeatferd, ytelsesforskjeller og kompleksitet gjennom animert eller konsollbasert visualisering.

### Funksjoner
- **Flere algoritmer**: Boble, Utvalg, Innsetting, Sammenslåing, Hurtigsort, etc.
- **Trinn-for-trinn visualisering**: Se hver sammenligning og bytte
- **Ytelsesmålinger**: Spor sammenligninger, bytter og utførelsestid
- **Sammenligningsmodus**: Kjør flere algoritmer på samme data
- **Tilpasset inndata**: Test med forskjellige array-størrelser og mønstre
- **Hastighetskontroll**: Juster visualiseringshastighet

### Tekniske detaljer
- **Språk**: C++
- **Visualisering**: Konsollbasert eller terminalgrafikk
- **Algoritmer**: Ulike sorteringsalgoritme-implementeringer
- **Målinger**: Tidskompleksitet og operasjonstelling

### Bruksområder
- Læring av sorteringsalgoritmer
- Algoritmeytelsesammenligning
- Undervisning i datastrukturer og algoritmer
- Praktisk forståelse av Big-O-notasjon
- Intervjuforberedelse
