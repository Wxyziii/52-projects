# Week 23 - Window Overlay Application

## English

### Overview
A window overlay application that displays information on top of other windows, similar to FPS counters, Discord overlays, or system monitors. Always visible regardless of active application.

### Features
- **Always on Top**: Window remains above all others
- **Transparency**: Semi-transparent background
- **Click-through**: Option to interact with windows below
- **Customizable Display**: Show custom information
- **Minimal Resource Usage**: Lightweight implementation
- **Draggable**: Reposition overlay as needed

### Technical Details
- **Language**: C++
- **GUI Framework**: Win32 API, X11, or Qt
- **Window Flags**: Topmost, transparent window attributes
- **Rendering**: Direct drawing or OpenGL

### Use Cases
- Gaming overlays (FPS, system stats)
- Development tools (performance monitoring)
- Screen annotations
- Learning GUI programming
- Understanding window management

## Norsk

### Oversikt
En vindusoverlegg-applikasjon som viser informasjon på toppen av andre vinduer, lik FPS-tellere, Discord-overlays eller systemmonitorer. Alltid synlig uavhengig av aktiv applikasjon.

### Funksjoner
- **Alltid på toppen**: Vinduet forblir over alle andre
- **Gjennomsiktighet**: Semi-transparent bakgrunn
- **Klikk-gjennom**: Opsjon for å samhandle med vinduer under
- **Tilpassbar visning**: Vis tilpasset informasjon
- **Minimal ressursbruk**: Lettvektsimplementering
- **Drabar**: Reposisjoner overlay etter behov

### Tekniske detaljer
- **Språk**: C++
- **GUI-rammeverk**: Win32 API, X11, eller Qt
- **Vindusflagg**: Topmost, transparente vindusattributter
- **Rendering**: Direkte tegning eller OpenGL

### Bruksområder
- Spill-overlays (FPS, systemstatistikk)
- Utviklingsverktøy (ytelsesovervåking)
- Skjermnotater
- Læring av GUI-programmering
- Forståelse av vindushåndtering
