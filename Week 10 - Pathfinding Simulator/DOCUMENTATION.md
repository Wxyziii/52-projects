# Week 10 - Pathfinding Simulator

## English

### Overview
A simulator that implements and visualizes pathfinding algorithms like A*, Dijkstra, BFS, and DFS. Shows how algorithms navigate from start to goal, avoiding obstacles and finding optimal paths on a grid or graph.

### Features
- **Multiple Algorithms**: A*, Dijkstra, BFS, DFS, Greedy Best-First
- **Grid Visualization**: See the algorithm explore the space
- **Obstacle Support**: Create mazes and blocked areas
- **Path Highlighting**: Show the final optimal path
- **Step-by-Step Execution**: Watch algorithm progression
- **Performance Metrics**: Compare nodes visited and path length

### Technical Details
- **Language**: C++
- **Data Structures**: Graphs, priority queues, sets
- **Visualization**: Console-based grid display
- **Heuristics**: Manhattan distance, Euclidean distance

### Use Cases
- Learning pathfinding algorithms
- Game AI development
- Robotics path planning
- Understanding graph traversal
- Algorithm optimization studies

## Norsk

### Oversikt
En simulator som implementerer og visualiserer veifinningsalgoritmer som A*, Dijkstra, BFS og DFS. Viser hvordan algoritmer navigerer fra start til mål, unngår hindringer og finner optimale veier på et rutenett eller graf.

### Funksjoner
- **Flere algoritmer**: A*, Dijkstra, BFS, DFS, Grådig Best-First
- **Rutenettvisualisering**: Se algoritmen utforske rommet
- **Hindringsstøtte**: Lag labyrinter og blokkerte områder
- **Veiutheving**: Vis den endelige optimale veien
- **Trinn-for-trinn-utførelse**: Se algoritmeprogresjonen
- **Ytelsesmålinger**: Sammenlign besøkte noder og veilengde

### Tekniske detaljer
- **Språk**: C++
- **Datastrukturer**: Grafer, prioritetskøer, mengder
- **Visualisering**: Konsollbasert rutenettvisning
- **Heuristikk**: Manhattan-avstand, Euklidsk avstand

### Bruksområder
- Læring av veifinningsalgoritmer
- Spill AI-utvikling
- Robotikk veplanlegging
- Forståelse av grafgjennomgang
- Algoritmeoptimaliseringsstudier
