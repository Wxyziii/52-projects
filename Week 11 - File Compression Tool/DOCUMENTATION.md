# Week 11 - File Compression Tool

## English

### Overview
A file compression and decompression utility that implements algorithms like Huffman coding or LZ77 to reduce file sizes. Demonstrates data compression techniques and bit-level manipulation.

### Features
- **File Compression**: Reduce file size using compression algorithms
- **File Decompression**: Restore original files perfectly
- **Multiple Algorithms**: Huffman coding, Run-Length Encoding, LZ77
- **Compression Ratio**: Display space savings achieved
- **Batch Processing**: Compress multiple files
- **Integrity Checking**: Verify decompressed data matches original

### Technical Details
- **Language**: C++
- **Algorithms**: Huffman coding, RLE, dictionary-based compression
- **Binary I/O**: Bit-level file reading and writing
- **Data Structures**: Binary trees, frequency tables

### Use Cases
- Learning compression algorithms
- Understanding information theory
- File backup and archiving
- Bandwidth-limited data transfer
- Storage optimization

## Norsk

### Oversikt
Et filkompressions- og dekomprimeringsverktøy som implementerer algoritmer som Huffman-koding eller LZ77 for å redusere filstørrelser. Demonstrerer datakomprimeringsteknikker og bit-nivå manipulering.

### Funksjoner
- **Filkomprimering**: Reduser filstørrelse ved hjelp av komprimeringsalgoritmer
- **Fildekomprimering**: Gjenopprett originale filer perfekt
- **Flere algoritmer**: Huffman-koding, Run-Length Encoding, LZ77
- **Komprimeringsforhold**: Vis plassbesparelser oppnådd
- **Batch-behandling**: Komprimer flere filer
- **Integritetskontroll**: Verifiser at dekomprimert data matcher original

### Tekniske detaljer
- **Språk**: C++
- **Algoritmer**: Huffman-koding, RLE, ordbokbasert komprimering
- **Binær I/O**: Bit-nivå fillesing og skriving
- **Datastrukturer**: Binære trær, frekvenstabell

### Bruksområder
- Læring av komprimeringsalgoritmer
- Forståelse av informasjonsteori
- Filsikkerhetkopiering og arkivering
- Båndbredde-begrenset dataoverføring
- Lagringsoptimalisering
