# Week 51 - Remote Administration Tool

## English

### Overview
A remote administration tool that allows secure remote control and management of computers over a network. Features remote desktop, file transfer, and system management capabilities for IT administration.

### Features
- **Remote Desktop**: View and control remote screen
- **File Transfer**: Upload/download files securely
- **Command Execution**: Run commands remotely
- **System Information**: View remote system details
- **Process Management**: Start/stop remote processes
- **Encrypted Communication**: Secure data transmission
- **Authentication**: Password or key-based access
- **Multi-client Support**: Manage multiple machines

### Technical Details
- **Language**: C++
- **Protocol**: Custom protocol over TCP/TLS
- **Screen Capture**: Platform-specific APIs
- **Compression**: Video/image compression
- **Encryption**: TLS/SSL for secure communication

### Use Cases
- IT remote support
- System administration
- Learning network programming
- Understanding remote access
- Building admin tools

## Norsk

### Oversikt
Et fjernadministrasjonsverktøy som tillater sikker fjernkontroll og administrasjon av datamaskiner over et nettverk. Inneholder eksternt skrivebord, filoverføring og systemadministrasjonsevner for IT-administrasjon.

### Funksjoner
- **Eksternt skrivebord**: Se og kontroller ekstern skjerm
- **Filoverføring**: Last opp/ned filer sikkert
- **Kommandoutførelse**: Kjør kommandoer eksternt
- **Systeminformasjon**: Se ekstern systemdetaljer
- **Prosesshåndtering**: Start/stopp eksterne prosesser
- **Kryptert kommunikasjon**: Sikker dataoverføring
- **Autentisering**: Passord eller nøkkelbasert tilgang
- **Flerklient-støtte**: Administrer flere maskiner

### Tekniske detaljer
- **Språk**: C++
- **Protokoll**: Tilpasset protokoll over TCP/TLS
- **Skjermfangst**: Plattformspesifikke APIer
- **Komprimering**: Video/bildekomprimering
- **Kryptering**: TLS/SSL for sikker kommunikasjon

### Bruksområder
- IT-fjernstøtte
- Systemadministrasjon
- Læring av nettverksprogrammering
- Forståelse av ekstern tilgang
- Bygging av admin-verktøy
