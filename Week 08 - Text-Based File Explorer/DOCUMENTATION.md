# Week 08 - Text-Based File Explorer

## English

### Overview
A command-line file explorer that allows users to navigate directories, view file information, and perform basic file operations without leaving the terminal. Provides a TUI (Text User Interface) for intuitive file management.

### Features
- **Directory Navigation**: Browse folders and subfolders
- **File Listing**: View files with details (size, date, permissions)
- **File Operations**: Copy, move, delete, rename files
- **Search Functionality**: Find files by name or pattern
- **File Preview**: View text file contents
- **Interactive Interface**: Keyboard-driven navigation

### Technical Details
- **Language**: C++
- **File System API**: Standard C++ filesystem library
- **Interface**: Console-based text interface
- **Platform**: Cross-platform compatible

### Use Cases
- Quick file management from terminal
- Server administration without GUI
- Learning filesystem APIs
- Understanding directory structures
- Lightweight alternative to graphical file managers

## Norsk

### Oversikt
En kommandolinje-filutforsker som lar brukere navigere i kataloger, se filinformasjon og utføre grunnleggende filoperasjoner uten å forlate terminalen. Gir et TUI (tekstbrukergrensesnitt) for intuitiv filbehandling.

### Funksjoner
- **Katalognavigering**: Bla gjennom mapper og undermapper
- **Filliste**: Vis filer med detaljer (størrelse, dato, tillatelser)
- **Filoperasjoner**: Kopier, flytt, slett, gi nytt navn til filer
- **Søkefunksjonalitet**: Finn filer etter navn eller mønster
- **Filforhåndsvisning**: Vis tekstfilinnhold
- **Interaktivt grensesnitt**: Tastaturbasert navigering

### Tekniske detaljer
- **Språk**: C++
- **Filsystem-API**: Standard C++ filesystem-bibliotek
- **Grensesnitt**: Konsollbasert tekstgrensesnitt
- **Plattform**: Kryssplattformkompatibel

### Bruksområder
- Rask filbehandling fra terminal
- Serveradministrasjon uten GUI
- Læring av filsystem-APIer
- Forståelse av katalogstrukturer
- Lettvektsalternativ til grafiske filbehandlere
