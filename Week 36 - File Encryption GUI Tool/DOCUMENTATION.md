# Week 36 - File Encryption GUI Tool

## English

### Overview
A graphical file encryption tool that provides a user-friendly interface for encrypting and decrypting files. Drag-and-drop support and visual progress indicators make security accessible to non-technical users.

### Features
- **Drag-and-Drop**: Easy file selection
- **Encryption/Decryption**: AES-256 encryption
- **Password Protection**: Secure file access
- **Progress Bar**: Visual encryption progress
- **Batch Operations**: Encrypt multiple files
- **File Shredding**: Secure deletion of originals
- **Key Management**: Save/load encryption keys

### Technical Details
- **Language**: C++
- **GUI Framework**: Qt or wxWidgets
- **Encryption**: AES-256-GCM or ChaCha20-Poly1305
- **Libraries**: OpenSSL or libsodium

### Use Cases
- Protecting sensitive files
- User-friendly encryption
- Learning GUI and crypto integration
- Building security applications
- Data privacy tools

## Norsk

### Oversikt
Et grafisk filkrypteringsverktøy som gir et brukervennlig grensesnitt for kryptering og dekryptering av filer. Dra-og-slipp-støtte og visuelle fremgangsindikatorer gjør sikkerhet tilgjengelig for ikke-tekniske brukere.

### Funksjoner
- **Dra-og-slipp**: Enkel filutvelgelse
- **Kryptering/dekryptering**: AES-256-kryptering
- **Passordbeskyttelse**: Sikker filtilgang
- **Fremgangsindikator**: Visuell krypteringsframgang
- **Batch-operasjoner**: Krypter flere filer
- **Filmakulering**: Sikker sletting av originaler
- **Nøkkelhåndtering**: Lagre/last krypteringsnøkler

### Tekniske detaljer
- **Språk**: C++
- **GUI-rammeverk**: Qt eller wxWidgets
- **Kryptering**: AES-256-GCM eller ChaCha20-Poly1305
- **Biblioteker**: OpenSSL eller libsodium

### Bruksområder
- Beskyttelse av sensitive filer
- Brukervennlig kryptering
- Læring av GUI og krypto-integrasjon
- Bygging av sikkerhetsapplikasjoner
- Datapersonvernverktøy
